
$ git clone https://github.com/https://github.com/pinkypatel2316/chatbot.git
$ cd chatbot
$ python3 -m venv venv
$ . venv/bin/activate
```
Install dependencies
```
$ (venv) pip install Flask torch torchvision nltk
```
Install nltk package
```
$ (venv) python
>>> import nltk
>>> nltk.download('punkt')
```

Run
```
$ (venv) python train.py






$ (venv) python chat.py
```
